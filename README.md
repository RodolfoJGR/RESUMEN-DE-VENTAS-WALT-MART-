# RESUMEN-DE-VENTAS-WALT-MART-
Resumen de ventas por metro cuadrado de una tienda departamental walt mart 
## Descripción
Proyecto de análisis de datos utilizando SQL y Python para identificar tendencias de ventas.

## Herramientas utilizadas

- Google Sheets

## Objetivos
- Limpiar datos
- Analizar tendencias
- Generar reportes

## Resultados
Se identificaron los productos más vendidos y los periodos de mayor demanda.
